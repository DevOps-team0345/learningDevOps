# learningDevOps
my devOps leaning journer with practival notes , commands , project, and CI/CD workflows...

## React project

A minimal React + Vite scaffold was added for learning and experimentation.

Setup:

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

## Docker

```bash
cp .env.example .env
docker compose build
docker compose up
```

The app will be available on http://localhost:3000.
